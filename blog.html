What is a set()?

Posted by Rahul on 12/24/2020

A set is a special data type in Python. More specifically, it is an unordered list of elements enclosed by {}. A set cannot have duplicate elements and if it does only one will show. In the following examples, I demostrated some common methods used with set operations. I used the add, update, remove and discard methods in the following examples.


s1 = {54, 67, 88, 88}
print(s1) # {88, 67, 54}
# to create an empty set, use set() not {}
s2 = set()
s2.add(23)
s2.add(34)
s2.add(53)
print(s2) # {34, 53, 23} <-- not in the same order as they were added
# not the case with lists
list_1 = []
list_1.append(23)
list_1.append(33)
print(list_1) # [23, 33]
s3 = {5, 5, 8}
s2.update(s3, [10, 11, 2])
# remove value of 5, but not 6 because 6 is not in s2
s2.remove(5)
print(s2)
# s2.remove(6) # KeyError: 6
s2.discard(6) # works just fine
print(s2) # {34, 2, 53, 23, 8, 10, 11}
set_1 = {25, 2, 1}
set_2 = {25, 2, 99}
set_3 = {25, 2, 1, 99}
print(set_1.intersection(set_2, set_3)) # {25, 2} <-- common in all sets
print(set_1.symmetric_difference(set_1)) # {} no values in set_1 that are not in itself
print(set_1.symmetric_difference(set_2)) # 1 is not in set_2 and 99 is not in set_1
# final note: the 'in' operator of Python 3 works faster with a set() than a list(), so
# be sure to convert a list() into a set() first when doing a membership test.

What is a deque()?

Posted by Rahul on 12/24/2020

A deque() is another data structure in Python that closely resembles a list(), but also has it own distinct characteristics. 'Deque' is a short-form for a "double-ended-queue" and the code below contains some common operations that one can perform on a deque() with methods as append, extend, pop, and rotate. pop, append, and extend also work with a list object in Python.

from collections import deque
# deque is a double-ended-queue
d = deque()
d.append(1)
d.append(22)
d.append(334) # deque([1,22,334])
print(d)
d.appendleft(334) # add 334 to the leftMost side: deque([334,1,22,334])
print(d)
d.popleft()
print(d) # and we're back to square one: deque([1,22,334])
d.extend([4, 5, 99]) # add all elements of list to right side of d
d.extendleft([4, 5, 55]) # add to the left of the d in this order: 55,5,4..
print(d) # d([55, 5, 4, 1, 22, 334, 4, 5, 99])
d.rotate(-3) # move everything in d 3 places to the left
print(d) # deque([1, 22, 334, 4, 5, 99, 55, 5, 4])
